# HPQC

This repository contains coursework for the High Performance and Quantum Computing module.

## Repository Structure

- `week2/` - Topic 2: Performance and Parallelism
- `week3/` - Topic 3: Introduction to MPI

Each weekly folder contains the code, notes, and instructions relevant to that topic.

## Week 2
Week 2 focuses on benchmarking program performance in C and Python, including:
- using the Linux `time` command
- internal timing within programs
- comparing interpreted and compiled code
- file input/output benchmarking

See `week2/README.md` for details.

## Week 3
Week 3 introduces MPI programming, including:
- MPI Hello World
- message passing between root and client processes
- serial and parallel vector operations
- comparing serial and parallel implementations

See `week3/README.md` for details.
